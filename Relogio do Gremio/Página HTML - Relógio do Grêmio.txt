Página HTML - Relógio do Grêmio
Este é o código HTML para uma página da web que exibe um relógio digital do Grêmio. A página contém elementos HTML para as horas, minutos e segundos, e inclui um arquivo CSS e um arquivo JavaScript externos para estilização e funcionalidade.

Elementos HTML
<!DOCTYPE html>: Declaração que define o tipo de documento HTML como HTML5.

<html lang="pt-br">: Elemento HTML raiz que define a linguagem da página como português brasileiro.

<head>: A seção do cabeçalho da página, que contém informações sobre o documento, como metadados, título e referências a arquivos externos.

<meta charset="UTF-8">: Define a codificação de caracteres da página como UTF-8 para suportar caracteres especiais.

<meta name="viewport" content="width=device-width, initial-scale=1.0">: Define as configurações de viewport para ajustar o layout da página em dispositivos móveis.

<link rel="stylesheet" href="assets/css/style.css">: Importa um arquivo CSS externo chamado "style.css" da pasta "assets/css" para estilizar a página.

<title>Relógio do Grêmio</title>: Define o título da página exibido na barra de título do navegador.

<body>: A seção do corpo da página, onde o conteúdo visível é colocado.

<div class="relogio">: Um contêiner div que envolve o relógio digital.

<div>: Três elementos div aninhados para exibir as horas, minutos e segundos do relógio.

<span id="horas">00</span>: Um elemento span com o ID "horas" usado para exibir as horas do relógio.

<span class="tempo">Horas</span>: Um elemento span com a classe "tempo" usado para rotular as horas.

<span id="minutos">00</span>: Um elemento span com o ID "minutos" usado para exibir os minutos do relógio.

<span class="tempo">Minutos</span>: Um elemento span com a classe "tempo" usado para rotular os minutos.

<span id="segundos">00</span>: Um elemento span com o ID "segundos" usado para exibir os segundos do relógio.

<span class="tempo">Segundos</span>: Um elemento span com a classe "tempo" usado para rotular os segundos.

<script src="assets/js/script.js"></script>: Importa um arquivo JavaScript externo chamado "script.js" da pasta "assets/js" para adicionar funcionalidade ao relógio.
